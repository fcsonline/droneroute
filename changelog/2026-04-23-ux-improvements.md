## Summary

UX improvements based on first-time user audit — better defaults, mobile fixes, and new map features.

## Changes

- Default waypoint altitude changed from 50m to 30m
- Default drone model changed from DJI M30 to DJI Mavic 3E
- Export KMZ button now shows a tooltip explaining why it's disabled
- Removed redundant "Add 2+ waypoints" text from the sidebar footer
- Fixed sidebar height on mobile using `h-dvh` so the sign-in button is always reachable
- Removed EGM96 (MSL) height reference option
- Elevation chart now shows a hint to drag waypoints for altitude adjustment
- Added satellite imagery layer (Esri) as a base layer option
- Changed guest sign-in prompt to "Guest mode. Sign in to save missions"
